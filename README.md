## üì± Tentang Aplikasi  
**Tiara Finance** adalah solusi digital all-in-one untuk manajemen keuangan lingkungan RT/RW secara transparan, akuntabel, dan modern. Aplikasi ini memudahkan warga untuk membayar iuran, melihat laporan keuangan real-time, dan berinteraksi antar warga, sekaligus memberikan alat yang powerful bagi pengurus (Ketua RT/Bendahara) untuk mengelola kas lingkungan.

Dibuat dengan **Flutter** untuk performa mobile yang maksimal dan desain antarmuka yang premium.

## üöÄ Cara Instalasi  

1. **Clone Repository**  
   ```bash
   git clone https://github.com/FikriBintangx/TIaraFinn.git
   cd TIaraFinn
   ```

2. **Install Dependencies**  
   ```bash
   flutter pub get
   ```

3. **Konfigurasi Firebase**  
   - Tambahkan file `google-services.json` (Android) dan `GoogleService-Info.plist` (iOS) ke dalam project.

4. **Jalankan Aplikasi**  
   ```bash
   flutter run
   ```
